Code Reference

Value Description View
.css() .css() 메서드는 선택한 요소에 CSS 속성 값을 설정합니다. View
.addClass() .addClass() 메서드는 선택한 요소에 클래스를 추가하는 메서드입니다. View
.removeClass() .removeClass() 메서드는 선택한 요소에 클래스를 삭제하는 메서드입니다. View
.text() .text() 메서드는 선택한 요소에 텍스트를 가져오거나 변경합니다. View
.mousemove() .mousemove() 메서드는 선택한 요소 범위 내에서 마우스를 움직였을 때 이벤트가 발생합니다. View
.hover() .hover() 메서드는 선택한 요소에서 마우스를 올렸을 때와 벗어 났을 때 각각 이벤트가 발생합니다. View
Value Description View
clientX 브라우저 페이지를 기준으로 X좌표의 위치를 반환합니다. View
clientY 브라우저 페이지를 기준으로 Y좌표의 위치를 반환합니다. View
offsetX 선택한 객체를 기준으로 X좌표의 위치를 반환합니다. View
offsetY 선택한 객체를 기준으로 Y좌표의 위치를 반환합니다. View
pageX 브라우저 페이지를 기준으로 X좌표의 위치를 반환합니다. View
pageY 브라우저 페이지를 기준으로 Y좌표의 위치를 반환합니다. View
screenX 모니터 스크린을 기준으로 X좌표의 위치를 반환합니다. View
screenY 모니터 스크린을 기준으로 Y좌표의 위치를 반환합니다. View

<script>
    //출력용
    $(window).mousemove(function (event) {
        $(".clientX").text(event.clientX);
        $(".clientY").text(event.clientY);
        $(".offsetX").text(event.offsetX);
        $(".offsetY").text(event.offsetY);
        $(".pageX").text(event.pageX);
        $(".pageY").text(event.pageY);
        $(".screenX").text(event.screenX);
        $(".screenY").text(event.screenY);
    });

    //마우스 움직이기
    $(window).mousemove(function (event) {
        $(".cursor").css({ left: event.clientX - 25, top: event.clientY - 25 });

        $(".contents em").hover(function () {
            $(".cursor").addClass("active");
        }, function () {
            $(".cursor").removeClass("active");
        });
    });
</script>
