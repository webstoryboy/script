Code Reference

Value Description View
.addClass() .addClass() 메서드는 선택한 요소에 클래스를 추가하는 메서드입니다. View
.removeClass() .removeClass() 메서드는 선택한 요소에 클래스를 삭제하는 메서드입니다. View
.text() .text() 메서드는 선택한 요소에 텍스트를 가져오거나 변경합니다. View
.mousemove() .mousemove() 메서드는 선택한 요소 범위 내에서 마우스를 움직였을 때 이벤트가 발생합니다. View
.hover() .hover() 메서드는 선택한 요소에서 마우스를 올렸을 때와 벗어 났을 때 각각 이벤트가 발생합니다. View
.position() .position() 메서드는 선택한 요소의 위치 좌표 값(기준점 기준)을 설정하거나 반환합니다. View
.width() .width() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다. View
.height() .height() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다. View
.fadeToggle() .fadeToggle() 메서드는 선택한 요소를 천천히 보여주고/숨깁니다. View
Value Description View
pageX 브라우저 페이지를 기준으로 X좌표의 위치를 반환합니다. View
pageY 브라우저 페이지를 기준으로 Y좌표의 위치를 반환합니다. View
clientX 브라우저 페이지를 기준으로 X좌표의 위치를 반환합니다. View
clientY 브라우저 페이지를 기준으로 Y좌표의 위치를 반환합니다. View

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.4.2/gsap.min.js"></script>
<script>
    let x = 0;
    let y = 0;

    $(".move-image").mousemove(function(e){
        gsap.to(".cursor", {duration: .2, left: e.pageX - 10, top: e.pageY - 10});

        //좌표값 중앙을 0으로 초기화
        //x = 컨텐츠 박스의 width()/2 + 왼쪽의 여백값
        //y = 컨텐츠 박스의 height()/2 + 윗쪽의 여백값

        // let x =  $(".move-image").width()/2;
        // let y =  $(".move-image").position().left;
        x =  e.clientX - ($(".move-image").width()/2 + $(".move-image").position().left);
        y =  e.clientY - ($(".move-image").height()/2 + $(".move-image").position().top);

        $(".img").attr({"style":"transform: translate(" + x/20 + "px," + y/20 + "px)" });
    });

    //출력용
    $(window).mousemove(function(e){ 
        $(".clientX").text(e.clientX);
        $(".clientY").text(e.clientY); 
        $(".mouseX").text(Math.floor(x)+"px");
        $(".mouseY").text(Math.floor(y)+"px"); 
    });

    //좌표 확인
    $(".cord").click(function(e){
        e.preventDefault();
        $(".cursor strong").fadeToggle();
    });
</script>
