Code Reference

Value Description View
.text() .text() 메서드는 선택한 요소에 텍스트를 가져오거나 변경합니다. View
.mousemove() .mousemove() 메서드는 선택한 요소 범위 내에서 마우스를 움직였을 때 이벤트가 발생합니다. View
.hover() .hover() 메서드는 선택한 요소에서 마우스를 올렸을 때와 벗어 났을 때 각각 이벤트가 발생합니다. View
.position() .position() 메서드는 선택한 요소의 위치 좌표 값(기준점 기준)을 설정하거나 반환합니다. View
.width() .width() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다. View
.height() .height() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다. View
.innerWidth() .innerWidth() 메서드는 선택한 요소의 가로(패딩 포함, 보더/마진 불포함)를 설정합니다. View
.innerHeight() .innerHeight() 메서드는 선택한 요소의 높이(패딩 포함, 보더/마진 불포함)를 설정합니다. View
.outerWidth() .outerWidth() 메서드는 선택한 요소의 가로(패딩/보더 포함, 마진 불포함)를 설정하거나 반환합니다. View
.outerHeight() .outerHeight() 메서드는 선택한 요소의 높이(패딩/보더 포함, 마진 불포함)를 설정하거나 반환합니다. View
Value Description View
innerWidth 브라우저의 콘텐츠 영역의 너비(스크롤 포함)값을 반환합니다. View
innerHeight 브라우저의 콘텐츠 영역의 높이(스크롤 포함)값을 반환합니다. View
outerWidth 브라우저의 툴바/스크롤바를 포함한 바깥 영역의 너비를 반환홥니다. View
outerHeight 브라우저의 툴바/스크롤바를 포함한 바깥 영역의 높이를 반환홥니다. View
Math.ceil() 숫자 보다 크거나 같은 수 중에서 가장 작은 정수를 반환합니다. View
Math.floor() 제공된 숫자 보다 작거나 같은 가장 큰 정수를 반환합니다. View
Math.round() 제공된 숫자를 반올림하여 가장 가까운 정수를 반환합니다. View

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.5.1/gsap.min.js"></script>
<script>    
    $(window).on("mousemove", mouseMove);

    function mouseMove(e){
        let postionSlow = (e.pageX - ($(window).width() / 2)) * 0.1;
        let postionFast = (e.pageX - ($(window).width() / 2)) * 0.2;

        //$(".spanSlow").css({"transform":"translateX("+ postionSlow +"px)"});
        //$(".spanFast").css({"transform":"translateX("+ -postionFast +"px)"});

        gsap.to(".spanSlow", { duration: 0.4, x: postionSlow });
        gsap.to(".spanFast", { duration: 0.4, x: -postionFast });

        //출력용
        $(".pageX").text(e.pageX);
        $(".pageY").text(e.pageY);
        $(".postionSlow").text(parseInt(postionSlow));
        $(".postionFast").text(parseInt(postionFast));        
    }
</script>
