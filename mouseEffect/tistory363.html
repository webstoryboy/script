Code Reference

Value Description View
innerWidth 브라우저의 콘텐츠 영역의 너비(스크롤 포함)값을 반환합니다. View
innerHeight 브라우저의 콘텐츠 영역의 높이(스크롤 포함)값을 반환합니다. View
outerWidth 브라우저의 툴바/스크롤바를 포함한 바깥 영역의 너비를 반환홥니다. View
outerHeight 브라우저의 툴바/스크롤바를 포함한 바깥 영역의 높이를 반환홥니다. View
Math.ceil() 숫자 보다 크거나 같은 수 중에서 가장 작은 정수를 반환합니다. View
Math.floor() 제공된 숫자 보다 작거나 같은 가장 큰 정수를 반환합니다. View
Math.round() 제공된 숫자를 반올림하여 가장 가까운 정수를 반환합니다. View

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.4.2/gsap.min.js"></script>
<script>
    const cursor = document.querySelector(".cursor");
    const width = cursor.offsetWidth;
    const height = cursor.offsetHeight;
    const rect = cursor.getBoundingClientRect();

    const follow = e => {
        gsap.to(cursor, 0.3, {left: e.pageX - rect.width/2, top: e.pageY - rect.height/2 });
    }

    document.addEventListener("mousemove", follow);
</script>
