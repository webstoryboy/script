Code Reference

Value Description View
.scrollTop() .scrollTop() 메서드는 선택한 요소의 세로 스크롤 값을 설정하거나 반환합니다. View
.scrollLeft() .addClass() 메서드는 선택한 요소의 가로 스크롤 값을 설정하거나 반환합니다. View
.offset() .addClass() 메서드는 선택한 요소의 위치 좌표 값(문서 기준)을 설정하거나 반환합니다. View
.width() .width() 메서드는 선택한 요소의 가로(패딩/마진/보더 불포함)를 설정합니다. View
.height() .height() 메서드는 선택한 요소의 높이(패딩/마진/보더 불포함)를 설정합니다. View
.text() .text() 메서드는 선택한 요소에 텍스트를 가져오거나 변경합니다. View
.addClass() .addClass() 메서드는 선택한 요소에 클래스를 추가하는 메서드입니다. View
.removeClass() .removeClass() 메서드는 선택한 요소에 클래스를 삭제하는 메서드입니다. View
.css() .css() 메서드는 선택한 요소에 CSS 속성 값을 설정합니다. View
.each() .hover() 메서드는 선택한 요소에 각각 함수를 실행합니다. View
Value Description View
parseInt() 문자형 데이터를 정수형 데이터로 바꿉니다. View
